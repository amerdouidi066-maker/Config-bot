# 🔥 THE ARCHITECT // ULTIMATE BOT (QWIKLABS EDITION)

بوت تيليجرام احترافي يقوم بأتمتة نشر خدمات VLESS على Google Cloud Run باستخدام المختبرات المؤقتة (Qwiklabs).

## الميزات
- **زر واحد فقط**: المستخدم يرسل رابط المختبر، والبوت يفعل الباقي.
- **تسجيل دخول تلقائي**: يستخدم بيانات `EMAIL` و `PASSWORD` المخزنة في متغيرات البيئة.
- **يعمل على المختبرات المؤقتة**: ينشئ حساب خدمة جديداً داخل كل مختبر وينشر الخدمة.
- **نظام طابور**: يعالج طلباً واحداً في كل مرة لتجنب حظر Google.
- **بدون تخزين**: لا يحفظ أي ملفات على الخادم (يُستخدم مجلد مؤقت).

## متغيرات البيئة المطلوبة
| المتغير | الوصف |
| :--- | :--- |
| `TOKEN` | توكن البوت من `@BotFather`. |
| `EMAIL` | البريد الإلكتروني لحساب Google المستخدم في المختبرات. |
| `PASSWORD` | كلمة المرور لحساب Google. |

## كيفية النشر على Railway
1. ارفع هذه الملفات على GitHub.
2. أنشئ مشروعاً جديداً على Railway واربط المستودع.
3. أضف المتغيرات البيئية الثلاثة (`TOKEN`, `EMAIL`, `PASSWORD`).
4. سيبدأ النشر تلقائياً.

## الاستخدام
1. أرسل `/start` للبوت.
2. اضغط على زر `🚀 تشغيل خدمة سريعة`.
3. أرسل رابط مختبر Google Skills (يحتوي على `project=`).
4. انتظر 2-3 دقائق – ستحصل على رابط VLESS.

## ملاحظة
هذا البوت صُمم خصيصاً للمختبرات المؤقتة (Qwiklabs). لا حاجة لمشروع دائم.
