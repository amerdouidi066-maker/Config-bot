# SHADOW LEGION v105.0 on Railway

## ⚙️ النشر السريع
1. ارفع هذه الملفات إلى GitHub.
2. في Railway -> New Project -> Deploy from GitHub.
3. أضف متغير البيئة: `TOKEN` = توكن البوت الخاص بك.
4. انتظر 5 دقائق وسيعمل البوت تلقائياً.

## 🛠️ الميزات
- نشر سحابة Cloud Run تلقائي
- أدوات اختراق حقيقية (Screenshot, DDoS, WiFi Stealer, ...)
- قاعدة بيانات SQLite مدمجة
- طابور انتظار للمهام

## ⚠️ ملاحظة
الأدوات التي تحتاج واجهة رسومية (Keylogger, Webcam) غير مدعومة في بيئة Railway لأنها لا تملك شاشة عرض.
